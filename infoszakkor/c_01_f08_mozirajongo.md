# Mozirajongó

## Probléma

Péter nagyon szeret moziba járni, és szeretné kiszámolni, hogy mennyi pénzt költ mozizásra egy évben.

## Feladat

Írj egy programot, ami kiszámítja az éves mozi költséget. A program kérdezze meg a felhasználótól, hogy átlagosan havonta hányszor jár moziba, mennyibe kerül egy mozijegy, valamint az éves mozi klub tagsági díj!

A program írja ki az éves költséget.

## Egy példa futás:
```
Input:  tagsagi dij = 3000
Input:  mozik szama havonta = 2
Input:  mozi jegy ara (Ft) = 1500
Output: eves koltseg: 39000 Ft
```
