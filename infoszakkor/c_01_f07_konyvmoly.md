# Könyvmoly

## Probléma

Hanna nagyon szeret olvasni, és szeretné nyomon követni, hogy mennyi pénzt költ könyvkölcsönzésre egy évben.

## Feladat

Írj egy programot, ami kiszámítja az éves könyvkölcsönzési költséget. A program kérdezze meg a felhasználótól, hogy átlagosan havonta hány könyvet kölcsönöz, és mennyi az egy könyv kölcsönzési díja, valamint az éves tagsági díj!

A program írja ki az éves költséget.

## Egy példa futás:
```
Input:  tagsagi dij = 2500
Input:	konyvek szama havonta = 3
Input:	konyvkolcsonzesi dij/konyv (Ft) = 200
Output:	eves koltseg: 9700 Ft
```
