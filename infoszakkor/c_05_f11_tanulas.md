# Tanulás időzítő

## Probléma

Egy diák szeretné hatékonyabban beosztani a tanulási idejét, ezért szeretne egy egyszerű időzítőt készíteni.

## Feladat

Írj egy programot, ami beolvassa a standard bemenetről a tanulni kívánt tantárgyak számát, a tanulási szakaszok közötti pihenőidőt (percben), majd minden tantárgyhoz bekéri a rá szánt tanulási időt (percben).

A program írja ki a standard kimenetre, hogy összesen hány percet kell tanulnia a diáknak aznap, a szünetekkel együtt.

Egy példa futás:
```
Input:	tantargyak szama = 3
Input:  szunet (perc) = 15
Input:	1. targy (perc) = 60
Input:	2. targy (perc) = 45
Input:	3. targy (perc) = 30
Output:	osszes tanulasi ido (perc): 165
```
