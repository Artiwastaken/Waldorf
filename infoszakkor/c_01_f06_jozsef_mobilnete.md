# Mobilnet

## probléma

József az egyik népszerű mobilszolgáltatónál tervez mobilinternet előfizetést vásárolni. A megfelelő csomag választásához szeretné megnézni, hogy a havi keretből hány videót tudna kedvenc tartalomgyártójától megnézni az adatkeretből.

## feladat

Írj egy programot, ami beolvassa a standard bemenetről a csomag adatkeretét (gigabájtban), valamint egy átlagos HD videó adatfelhasználását (gigabájtban).

A program írja ki a standard kimenetre, hogy hány teljes videót tudna megnézni József a hónapban.

## egy példa futás:
```
Input:	adatkeret (GB) = 5
Input:	atlag video adatigenye (GB) = 0.3
Output:	teljes videok : 16
```
