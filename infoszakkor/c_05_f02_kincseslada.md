## probléma

Találtunk egy számzáras ládát egy lakatlan szigeten, benne a túléléshez szükséges eszközökkel. A társunk a sziget túloldaláról hozott vissza egy papírcetlit, amin a következő szöveg áll:

"Ki ezeket a sorokat olvasod, add össze az összes háromjegyű, 3-mal vagy 5-tel osztható természetes számot és minden kívánságod teljesül."


## feladat

Írj egy függvényt, ami visszatér a számzár kódjával. A főprogramban a függvény meghívásával írd a standard outputra a számzár kódját.

Példa futás:
```
Std. Input:	-

Std. Output:
a szamzar kodja: 230850
```
