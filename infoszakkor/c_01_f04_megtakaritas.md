# probléma

Márk pénzt gyűjt, hogy új telefont vegyen. Úgy dönt, hogy minden héten egy fix összeget félretesz. Szeretné tudni, mennyi pénze lesz egy bizonyos hét után.

## feladat
Írj egy programot, ami beolvassa a standard bemenetről a hetente megtakarított összeget és az eltelt hetek számát.

A program írja ki a standard kimenetre, hogy Márknak mennyi pénze lesz az eltelt hetek után.

Egy példa futás:
```
Input:	hetente megtakaritott osszeg (ft) = 4000
Input:	eltelt hetek szama = 24
Output:	megtakaritas 24 het utan (ft): 96000
```
