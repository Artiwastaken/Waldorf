# Áruházi hűségprogram

## Probléma

Egy nagyvárosi áruházlánc az új vásárlói hűségprogramját teszteli. Ezer vásárló bemutatkozó kedvezményét rögzítették egy adott napon. A kedvezmények 0 és 70 közötti egész számok, a teljes vásárlási érték százalékában megadva.

## Feladat

Írj egy programot, amely véletlenszámok előállításával szimulálja az ezer vásárlóra rögzített bemutatkozó kedvezményértékeket. A program írja a standard kimenetre azoknak a vásárlóknak a sorszámát és a kedvezmény mértékét, akiknek a kedvezménye módosításra szorul. A 10% alatti kedvezményeket kell megnövelni, mivel ezek a programban túl alacsonynak minősülnek.

A kiírás az áruházlánc rendszere miatt a legutolsó vásárlótól a legelső felé történjen!

Példa futás:
```
Kedvezmenyek felulvizsgalata:
MODOSITAS	529. vasarlo:	8% kedvezmeny
MODOSITAS	344. vasarlo:	3% kedvezmeny
MODOSITAS	12. vasarlo:	0% kedvezmeny
```
