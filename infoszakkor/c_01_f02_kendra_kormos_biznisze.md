# probléma

Kendra műkörmösként dolgozik. A vállalkozása eléggé népszerű, de nehezen tartja számon pénzügyeit. Felbérelt ezért téged, hogy könyveld a kiadásait és bevételeit az évre vonatkozóan.

Az összesített számok birtokában nincs más hátra, mint meghatározni hogy mekkora veszteséggel vagy profittal zárult Kendra éve.


## feladat

Írj egy programot, ami beolvassa a standard bemenetről az egész éves anyagköltséget, rezsiköltséget és bevételt.

A program írja ki a standard kimenetre, hogy Kendra milyen végösszeggel zárta az évet. (a pozitív eredmény bevételt, a negatív eredmény veszteséget jelent)

Egy példa futás:
```
Input:	eves anyagkoltseg (ft) = 275000
Input:	eves rezsikoltseg (ft) = 210000
Input:	eves bevetel (ft) = 1130000
Output:	eves veszteseg/profit: 645000
```
