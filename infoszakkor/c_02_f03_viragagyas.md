## probléma

Mariann egy panelházban lakik, de nagyon szeret kertészkedni. Kitalálta, hogy a ház előtt található kert egy háromszög alakú részét szeretné beültetni.

Még nem döntötte el pontosan mekkora összeget tud a növényekre, virágokra és tápszerekre költeni.

A kalkulációhoz írunk neki egy programot, amellyel megnézheti mekkora területet fedne le egy tetszőleges háromszög alakú rész a kertből.

## feladat

Írj egy függvényt, amelynek paraméterei egy a kertből egy tetszőleges háromszög a, b és c oldalának hossza törtszámként, méterben megadva.

A függvény térjen vissza a kertrész területével.

A főprogramban teszteljük a függvény működését.

Egy példa futás:
```
Input:  Kertresz oldalai meterben= 3.0 4.0 5.0
Output: Viragagyas merete: 6.000000
```
