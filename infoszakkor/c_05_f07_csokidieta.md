## probléma

Kati diétázik, de kapott ajándékba egy doboz Ferrero Rocher-t. Egy csokoládéban 75 kalória van. Segíts Katinak kiszámolni, hogy a megadott napon hány csokoládét fogyaszthat.

## feladat

Írj egy függvényt, amely megkapja paraméterként az adott napi, nasizásra szánt kalórialimitet (egész szám).

A függvény írja ki a standard outputra, hogy csokinként mennyivel növekszik Kati napi kalória bevitele, egészen addig ameddig el nem éri a maximálisan elfogyasztható kalóriák számát. Ha a következő csoki már átlépné a limitet, akkor azt a csokit Kati már nem fogja elfogyasztani.


Példa futás:
```
Std. Input:
Napi nasi kaloria limit: 400

Std. Output:
elfogyasztott csokik: 1, ez osszesen 75 kaloria
elfogyasztott csokik: 2, ez osszesen 150 kaloria
elfogyasztott csokik: 3, ez osszesen 225 kaloria
elfogyasztott csokik: 4, ez osszesen 300 kaloria
elfogyasztott csokik: 5, ez osszesen 375 kaloria
```

Példa futás 2:
```
Std. Input:
Napi nasi kaloria limit: 300

Std. Output:
elfogyasztott csokik: 1, ez osszesen 75 kaloria
elfogyasztott csokik: 2, ez osszesen 150 kaloria
elfogyasztott csokik: 3, ez osszesen 225 kaloria
elfogyasztott csokik: 4, ez osszesen 300 kaloria
```
